No código há comentários explicando cada parte dele, porém de forma geral são duas classes (book e BookList) no qual uma cria livros e a outra uma lista de livro. Na classe book, é necessário entrar com informações do nome do livro, autor, gênero e se ele já foi lido ou não.

Na classe BookList, existe 4 métodos no qual:

- addBook - adiciona o livro na lista;
- setNextBook - determina qual o próximo livro que será lido;
- startCurrentBook - seleciona o livro que deve começar a ser lido;
- finishCurrentBook - determina que o livro que estava sendo lido foi terminado.
